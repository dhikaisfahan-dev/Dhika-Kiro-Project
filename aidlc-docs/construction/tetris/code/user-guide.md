# Tetris Game - User Guide

## Welcome to Tetris!

This is a comprehensive web-based Tetris game featuring modern mechanics, multiple game modes, power-ups, achievements, and more. This guide will help you get started and master the game.

## Getting Started

### Opening the Game
1. Open `tetris/index.html` in a modern web browser
2. Enter your player name
3. Select difficulty level (Easy, Medium, or Hard)
4. Choose a game mode (Marathon, Sprint, or Ultra)
5. Click "Start Game" to begin

### First-Time Setup
- The game will ask for your name on first launch
- Your name will be saved for future sessions
- You can change your name in the settings menu

## Game Controls

### Keyboard Controls
- **Left Arrow**: Move piece left
- **Right Arrow**: Move piece right
- **Down Arrow**: Soft drop (faster falling)
- **Up Arrow**: Rotate piece clockwise
- **Z Key**: Rotate piece counter-clockwise
- **Spacebar**: Hard drop (instant placement)
- **C Key**: Hold current piece
- **P Key**: Pause game
- **Escape**: Pause game

### Control Tips
- Hold down arrow keys for continuous movement
- Use hard drop (Spacebar) for quick placement and bonus points
- Hold piece (C) can only be used once per piece
- Practice rotating pieces near walls to learn wall kicks

## Game Modes

### Marathon Mode
**Goal**: Survive as long as possible and maximize your score

- Start at Level 1
- Level increases every 10 lines cleared
- Speed increases with each level
- Game continues until you reach the top
- Best for: Learning the game, high score challenges

### Sprint Mode
**Goal**: Clear 40 lines as fast as possible

- Timer counts up from 0:00
- Game ends when 40 lines are cleared
- Leaderboard shows best times
- Best for: Speed challenges, improving efficiency

### Ultra Mode
**Goal**: Score as many points as possible in 3 minutes

- Timer counts down from 3:00
- Game ends when time runs out
- Maximize score with combos and special moves
- Best for: Quick games, score optimization

## Scoring System

### Basic Scoring
- **Single Line**: 100 × level points
- **Double Line**: 300 × level points
- **Triple Line**: 500 × level points
- **Tetris (4 lines)**: 800 × level points
- **Soft Drop**: 1 point per cell
- **Hard Drop**: 2 points per cell

### Combo System
- Clear lines consecutively to build combos
- **Combo Bonus**: 50 × combo count × level points
- Combo resets if no lines are cleared
- Higher combos = higher scores!

### Back-to-Back Bonus
- Clear Tetris or T-spin consecutively
- **Bonus**: 1.5× multiplier on the second clear
- Maintains until you make a non-difficult clear
- Stack with combos for massive scores!

### T-Spin Scoring
- **T-spin Mini**: 100 × level points
- **T-spin Single**: 800 × level points
- **T-spin Double**: 1200 × level points
- **T-spin Triple**: 1600 × level points
- T-spins are advanced moves - practice to master!

## Modern Tetris Features

### Ghost Piece
- Semi-transparent piece showing where current piece will land
- Helps plan placement
- Updates as you move and rotate
- Matches your current theme colors

### Hold Piece
- Press C to hold current piece
- Swap held piece with current piece
- Can only use once per piece
- Strategic tool for saving pieces

### Hard Drop
- Press Spacebar for instant placement
- Awards 2 points per cell dropped
- Fastest way to place pieces
- Essential for high-level play

### Wall Kicks
- Pieces can rotate near walls and blocks
- Uses Super Rotation System (SRS)
- Allows advanced maneuvers
- Practice to learn kick patterns

## Power-ups

Power-ups appear randomly during gameplay (5% chance per piece). Collect them by landing a piece on them!

### Clear Row
- **Effect**: Instantly clears the bottom row
- **Duration**: Instant
- **Strategy**: Use when bottom is cluttered

### Slow Time
- **Effect**: Reduces fall speed by 50%
- **Duration**: 30 seconds
- **Strategy**: Great for planning complex moves

### Bomb Piece
- **Effect**: Clears 3×3 area on placement
- **Duration**: Next piece only
- **Strategy**: Place in crowded areas

### Line Blast
- **Effect**: Clears all incomplete lines
- **Duration**: Instant
- **Strategy**: Emergency cleanup tool

### Ghost Mode
- **Effect**: Next 5 pieces pass through blocks
- **Duration**: 5 pieces
- **Strategy**: Recover from high stacks

## Tips and Strategies

### Beginner Tips
1. **Keep it flat**: Avoid creating tall columns
2. **Save the I-piece**: Use hold for I-pieces when needed
3. **Watch the ghost**: Use ghost piece to plan placement
4. **Clear lines early**: Don't let blocks stack too high
5. **Practice rotating**: Learn piece rotations

### Intermediate Tips
1. **Build for Tetrises**: Create 4-line gaps for big scores
2. **Use combos**: Clear lines consecutively
3. **Learn T-spins**: Practice T-spin setups
4. **Manage hold piece**: Strategic piece saving
5. **Plan ahead**: Look at next pieces

### Advanced Tips
1. **Master T-spins**: Learn all T-spin patterns
2. **Maintain back-to-back**: Chain difficult clears
3. **Optimize speed**: Use hard drops efficiently
4. **Stack strategically**: Create combo opportunities
5. **Adapt to speed**: Stay calm at high levels

### Power-up Strategies
1. **Save Slow Time**: Use during difficult situations
2. **Bomb strategically**: Clear problem areas
3. **Ghost Mode recovery**: Use when stack is too high
4. **Line Blast timing**: Use before game over
5. **Clear Row wisely**: Remove cluttered bottom rows

## Troubleshooting

### Game Won't Start
- Ensure JavaScript is enabled
- Try a different browser
- Clear browser cache
- Check browser console for errors

### No Audio
- Click anywhere to enable audio (browser requirement)
- Check mute button isn't active
- Verify volume slider isn't at 0
- Check browser audio permissions

### Performance Issues
- Close other browser tabs
- Update your browser
- Try a different browser
- Reduce browser zoom level

### Controls Not Working
- Click on game area to focus
- Check keyboard isn't locked
- Try refreshing the page
- Verify browser supports keyboard events

### Scores Not Saving
- Check localStorage is enabled
- Verify browser isn't in private mode
- Clear old data if storage is full
- Try a different browser

## Frequently Asked Questions

### Q: Can I play on mobile?
A: The game is responsive and works on mobile, but keyboard controls are required. Touch controls are not yet implemented.

### Q: How do I unlock all achievements?
A: Play different game modes and try various strategies. Some achievements require specific conditions or extended play time.

### Q: Can I play with friends in real-time?
A: Currently only asynchronous multiplayer (score comparison) is supported. Real-time multiplayer is not available.

### Q: Where are my scores saved?
A: Scores are saved in your browser's localStorage. They persist across sessions but are local to your browser.

### Q: How do I reset my progress?
A: Use the "Reset Data" option in settings. Warning: This clears all scores, achievements, and preferences!

### Q: What browsers are supported?
A: Modern browsers with HTML5 Canvas and Web Audio API support: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+.

### Q: Why is there no sound?
A: Audio files are placeholders. You'll need to add actual audio files to the `assets/audio/` directory.

### Q: Can I customize the controls?
A: Custom key bindings are not currently supported. The default keyboard controls are fixed.

### Q: How do T-spins work?
A: T-spins are special moves where you rotate a T-piece into a tight space. They award bonus points and are key to high scores.

### Q: What's the highest possible score?
A: There's no theoretical limit in Marathon mode. Scores depend on level reached, combos achieved, and special moves performed.
